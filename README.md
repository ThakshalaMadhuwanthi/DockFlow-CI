#DockFlow-CICD
# DevOps Pipeline 🚀

A simple Node.js application containerized with Docker and deployed using GitHub Actions CI/CD.

## 🔧 Setup

- Node.js app serving a simple web page
- Docker used for containerization
- GitHub Actions used for automation
- Self-hosted runner configured on VirtualBox VM

## ⚙️ CI/CD Pipeline

✔ Builds Docker image  
✔ Pushes image to Docker Hub  
✔ Pulls image on self-hosted runner (VirtualBox VM)  
✔ Deploys and runs container automatically
